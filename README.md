# 🚀 Portfólio Profissional | Leonardo Gomes de Queiros

Bem-vindo ao repositório do meu portfólio profissional. Este projeto foi desenvolvido para centralizar minhas principais entregas e competências técnicas, atuando como um cartão de visitas digital otimizado tanto para dispositivos móveis quanto para desktop.

O projeto está publicado e configurado exclusivamente através do **GitHub Pages**.

---

## 🎯 Sobre o Portfólio

Este é um portfólio de página única (*Single Page Application* simplificada) projetado com foco em **alta performance**, **design moderno** e **experiência do usuário (UX)**. O layout utiliza um tema escuro e elementos futuristas com tons de ciano, roxo e azul-ardósia.

### ✨ Funcionalidades e Diferenciais
* **Design 100% Responsivo**: Adaptação fluida para smartphones, tablets e desktops utilizando Bootstrap 5.
* **Animações ao Rolar (AOS)**: Efeitos dinâmicos de *fade-in*, *zoom* e transições suaves que tornam a leitura do perfil mais engajadora.
* **Navegação Fluida (Scrollspy)**: Menu superior inteligente que acompanha a rolagem da página e destaca automaticamente a seção ativa.
* **Destaque de Métricas Corporativas**: Ênfase em dados de grande relevância e premiações conquistadas no mercado financeiro.

## 🌐 Como Publicar no GitHub Pages

Como este repositório foi estruturado exclusivamente para funcionar como uma **GitHub Page**, siga os passos abaixo para colocá-lo no ar:

1.  Faça o **Push** do arquivo `index.html` para a branch principal (ex: `main` ou `master`).
2.  No seu repositório no GitHub, clique na aba **Settings** (Configurações).
3.  No menu lateral esquerdo, navegue até a seção **Pages**.
4.  Em **Build and deployment**, sob a opção *Source*, selecione **Deploy from a branch**.
5.  Escolha a sua branch principal (ex: `main`) e a pasta raiz (`/(root)`).
6.  Clique em **Save**.
7.  Aguarde alguns minutos e o GitHub fornecerá a URL pública do seu portfólio (ex: `https://leonardo-queiros.github.io/praticas-profissionais/`).
